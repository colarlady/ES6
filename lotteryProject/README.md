## 总结

+ ES6默认是严格模式，注意作用域的问题
+ 使用类和对象，结合模块化构建大型项目
+ 使用Promise替换传统的回调函数
+ 使用箭头函数需要注意this的指向
+ 学会使用generator完成异步操作应用
